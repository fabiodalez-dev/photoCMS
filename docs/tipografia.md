## Panoramica Sistema

Cimaise gestisce 4 contesti tipografici indipendenti:

1. **Headings** (Titoli) - h1, h2, h3, titoli album, titolo sito
2. **Body** (Corpo testo) - Paragrafi, descrizioni, testi lunghi
3. **Captions** (Didascalie) - Didascalie immagini, metadati EXIF
4. **Navigation** (Navigazione) - Menu, link, pulsanti

Per ogni contesto puoi scegliere:
- **Font family** (40+ font disponibili)
- **Font weight** (peso: 400, 500, 600, 700)

## Contesti Tipografici

### 1. Headings (Titoli)

**Applica a**:
- Titolo sito nell'header
- Titoli album (h1)
- Titoli pagina (h2, h3)
- Sezioni (`<h4>`, `<h5>`, `<h6>`)

**Default**:
- Font: **EB Garamond**
- Weight: **600** (Semi-Bold)

**Consiglio**:
- Usa serif per eleganza editoriale

**Esempi**:

## Come Scegliere i Font

### Step 1: Definisci lo Stile del Portfolio

**Editoriale / Fine Art**:
- Headings: Serif classico (Playfair, EB Garamond)
- Body: Serif leggibile (Crimson Text, Lora)

**Moderno / Minimal**:
- Headings: Sans-serif geometrico (Inter 600, Space Grotesk)
- Body: Sans-serif pulito (Inter 400, DM Sans)

**Fashion / Luxury**:
- Headings: Serif alto contrasto (Bodoni Moda, Abril Fatface)
- Body: Sans-serif elegante (Manrope, Plus Jakarta Sans)

**Documentary / Reportage**:
- Headings: Sans-serif bold (Archivo, Urbanist)
- Body: Sans-serif leggibile (Noto Sans, Source Sans 3)

---

### Step 2: Scegli un Abbinamento

**Regola d'oro**: Contrasto o Coerenza

#### **Contrasto** (Serif + Sans-Serif)
Crea gerarchia visiva chiara.

```
Headings: Playfair Display 700 (serif)
Body:     Inter 400 (sans-serif)
‚Üí Elegante ma moderno
```

#### **Coerenza** (Tutto Serif o Tutto Sans)
Crea armonia visiva.

```
Headings: Inter 700 (sans-serif)
Body:     Inter 400 (sans-serif)
‚Üí Minimal e pulito
```

---

### Step 3: Testa con Contenuti Reali

1. Vai su **Admin ‚Üí Impostazioni ‚Üí Tipografia**
2. Scegli font per Headings
3. **Preview Live** mostra subito il cambiamento
4. Prova 3-4 combinazioni
5. Visita frontend per vedere su album reali
6. Salva quando sei soddisfatto

**Tip**: Apri il tuo portfolio in 2 finestre affiancate e confronta combinazioni diverse.

## Personalizzazione Avanzata

### CSS Custom per Font Size

Il sistema tipografico imposta font-family e weight. Per controllare dimensioni:

**Admin ‚Üí Impostazioni ‚Üí Frontend ‚Üí Custom CSS**

```css
/* Ingrandisci titoli album */
.album-title {
  font-size: 3rem !important; /* default: 2.5rem */
}

/* Riduci body text su mobile */
@media (max-width: 768px) {
  body, p {
    font-size: 0.95rem !important; /* default: 1rem */
  }
}

/* Didascalie pi√π piccole */
.image-caption, .exif-data {
  font-size: 0.8rem !important; /* default: 0.875rem */
}

/* Menu uppercase */
nav a, .nav-link {
  text-transform: uppercase;
  letter-spacing: 0.05em;
}
```

### Caricamento Font Custom (Avanzato)

Se hai font acquistati (es. da MyFonts, Adobe Fonts):

1. **Converti in WOFF2** (formato web moderno):
   - Usa [Transfonter](https://transfonter.org/)
   - Upload .ttf/.otf
   - Scarica .woff2

2. **Carica in `/public/fonts/custom/`**

3. **Aggiungi @font-face via Custom CSS**:

```css
@font-face {
  font-family: 'MioFont Custom';
  src: url('/fonts/custom/miofont-regular.woff2') format('woff2');
  font-weight: 400;
  font-display: swap;
}

@font-face {
  font-family: 'MioFont Custom';
  src: url('/fonts/custom/miofont-bold.woff2') format('woff2');
  font-weight: 700;
  font-display: swap;
}

/* Applica a headings */
h1, h2, h3, .site-title {
  font-family: 'MioFont Custom', serif !important;
}
```

## Font Loading e Performance

### Font-Display: Swap

Tutti i font Cimaise usano `font-display: swap` per evitare FOIT (Flash Of Invisible Text).

**Comportamento**:
1. Pagina carica con font di sistema (immediato)
2. Font custom scarica in background
3. Swap automatico quando pronto (0-3 secondi)

**Risultato**: Nessun testo invisibile, ottima UX.

---

### Preload Font Critici (Opzionale)

Per eliminare anche il flash di cambio font, aggiungi preload nel Custom CSS:

```html
<!-- In: Admin ‚Üí Settings ‚Üí Custom CSS ‚Üí sezione <head> se disponibile -->
<link rel="preload" href="/fonts/playfair-display/playfair-display-700.woff2" as="font" type="font/woff2" crossorigin>
```

**Solo per font above-the-fold** (headings visibili subito).

---

### Lazy Loading Font Secondari


## FAQ Tipografia

### Posso usare Google Fonts?

Cimaise serve tutti i font **localmente** per GDPR compliance. Non si collega a Google Fonts CDN.

I font disponibili sono gi√† stati scaricati da Google Fonts e ottimizzati.

### Quanti font dovrei usare?

**Consiglio**:
- **1 font** (tutto sans-serif o tutto serif) ‚Üí Coerenza massima
- **2 font** (serif + sans-serif) ‚Üí Gerarchia visiva chiara
- **3+ font** ‚Üí ‚ùå Rischio caos visivo

### Posso cambiare font solo per alcune pagine?

Non via UI. Richiede Custom CSS con selettori specifici:

```css
/* Solo pagina About */
body.page-about h1 {
  font-family: 'Cinzel', serif !important;
}

/* Solo album categoria "Wedding" */
.album-category-wedding .album-title {
  font-family: 'Playfair Display', serif !important;
}
```

### I font influenzano le performance?

Minimamente. Cimaise carica **solo i pesi effettivamente usati**.

**Esempio**:
- Usi Inter 400, 500, 600 ‚Üí Carica 3 file (~60KB totali)
- NON carica Inter 300, 700, 800, 900

**Ottimizzazione**:
- WOFF2 compression (miglior compressione)
- `font-display: swap` (no blocking)
- Subset latino (no caratteri cinesi/arabi se non usati)
