Cimaise Installation

Step 1: Environment Configuration

Create a .env file in your project root directory with your database configuration:

DB_HOST=localhost
DB_NAME=your_database_name
DB_USER=your_database_user
DB_PASS=your_database_password
APP_URL=https://yourdomain.com

Step 2: Database Setup

Create the required database tables by running the SQL migrations provided in the /migrations directory.

Step 3: Permissions

Ensure the web server has write permissions to the following directories:

  • /public/media
  • /cache (if it exists)
  • /logs (if it exists)
Note: After completing these steps, refresh this page to continue with the installation.
